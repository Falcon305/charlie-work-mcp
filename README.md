# Charlie Work

> *"This is Charlie Work. Nobody else will do it. That's why it's yours."*

An MCP server that surfaces — and dignifies — the **toil** in your repo: the un-fun, load-bearing
maintenance work everyone ignores until it bites. Flaky tests rat-nesting in CI. A TLS cert nine
days from death. Dead feature flags. TODO rot. Runbooks nobody owns.

It scans your repo, hands you a prioritized queue, and keeps a **credit ledger** so the invisible
work becomes visible in standup. Named after the episode where the gang discovers Charlie has been
quietly holding Paddy's together the whole time.

Serious mode is one flag away — the jokes are a toggle, never a tax.
